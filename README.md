# Smart Dustbin - Waste Management System

A Smart Waste Management System developed to automate waste monitoring and improve waste collection efficiency using modern web technologies and smart hardware integration.

This project combines a responsive frontend, backend services, and smart dustbin technology to provide real-time monitoring and management of waste levels.

---

# Project Overview

The Smart Dustbin system is designed to solve problems related to inefficient waste management by providing:

- Real-time waste level monitoring
- Smart automation
- Improved cleanliness management
- Web-based monitoring dashboard
- Efficient waste collection support

The system can be integrated with sensors and microcontrollers for smart city and IoT-based applications.

# Installation Guide

## Step 1: Clone the Repository

```bash
git clone https://github.com/Aditi7105/Smart_Dustbin.git
```

---

## Step 2: Navigate to the Project Folder

```bash
cd Smart_Dustbin
```

---

# Frontend Setup

## Navigate to Frontend Folder

```bash
cd frontend
```

## Install Dependencies

```bash
npm install
```

## Start Frontend Server

```bash
npm start
```

---

# Backend Setup

## Navigate to Backend Folder

```bash
cd backend
```

## Install Dependencies

```bash
npm install
```

## Start Backend Server

```bash
npm start
```
